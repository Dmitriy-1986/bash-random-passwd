## Генератор паролей в командной строке Linux

- Установить утилиту **pwgen**:
` sudo apt install pwgen `
- Сделать файл исполняемым:
` chmod +x script.sh  `
-  Запустить скрипт:
` ./script.sh `
---
Ключ -y добавляет к стандартному набору символов знаки препинания и специальные символы
` pwgen 10 4 -y `
